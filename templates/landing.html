OBJECTIVE AND SOLUTION

One stop solution for text analysis

We don't let weak English become a barrier for academic studies.

In most places, english is the primary medium of instruction. If someone is not fluent in Englishlanguage they have to suffer academically, which is very unfortunate, and language shouldn't be something which restricts education.

Utilised strong Machine learning models for performing multiple tasks

Nowadays teachers and students alike have a lot on their plate, with advancing technology, our application utilises the power of strong machine learning models developed for natural language processing to help them in their daily schedule

Generate questions for assignments and predict answers

Generating questions for assignments and unseen passages is a monotonous task which can be now automated, and our application aims to do just that

Paraphrase

Generate paraphrased text, questions, essays etc to improve the quality of writing and fluency.

  • Paraphrased text generated using strong NLP models
  • Useful for students to imrpove the fluency of essays and answers to increase their grades.
  • Removes the barrier of not being fluent in english language.

Summarize

Generate a summary of the text entered. Summary generated using multiple NLP models like bag of words etc.

  • Helps students get a crux of the study material
  • Allows teachers to provide study notes with less hassle
  • Provides summary in easy and understandable english, great for last minute preparation

Question Generator

Enter text data and generate questions for the same. More questions generated for more data.

  • Two options for question generation are available, Boolean and MCQ type questions
  • One correct option is present along with 3 incorrect ones. Advanced Machine learning models generate the wrong operations.
  • Highly useful to teachers for generating assignments and quiz questions

Answer Predictor

Enter the text data and question and get predicted answers based on the provided context.

  • Select the type of answer you want, Boolean or detailed/MCQ and get predicted answers
  • Answers predicted by advanced machine learning models.
  • Can be used by students for generating answers to assignments and questions and get an idea about how to solve.

Text to Speech

Converts live text to speech, can be used for reading out loud essays, assignments and study material.

  • Read out loud the text entered
  • Available for multiple languages, not only English. SOme available languages include Spanish, German, etc.
  • Option available to download the audio file generated for later use.

Speech to text

Real time speech converted to text. Can be copied and used directly for a variety of purposes

  • Generates highly accurate text results from speech using API to give the best results.
  • Very helpful for making notes from speech when unable to write or type
  • Has an option to copy the text for direct use at other placed like generating summary etc.
