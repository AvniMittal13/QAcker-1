Features offerred

With our web application you can perform multiple operations on textual data

SUMMARIZE

Enter the text data and generate summary of the same using multiple NLP models

PARAPHRASE

Enter the text data and generate paraphrased version of the same. Very useful to improve the quality of assignments, essays, questions etc.

QUESTION GENERATOR

Enter the context and choose the type of question you'd like to generate : Boolean (Yes/No) or MCQ type. Based on the amount of context provided, you can get different questions.

ANSWER PREDICTOR

Enter the context and the question whose answer you'd like to find. Select weather you want a Boolean(YES/NO) or detailed answer. The appliaction analyses the data provided and predicts the answer woth high accuracy.

TEXT TO SPEECH

Make the web application read aloud the text entered by you. It can read aloud the various summaries generated as well.

REAL TIME SPEECH TO TEXT

Generate real time text from speech, can be very useful when you don't want to type or write and gather your thoughts at one place.
